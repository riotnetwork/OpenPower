Maxim 78M6613 dual hase energy monitor Driver for the Arduino platform
Provides high resolution energy monitoring for Arduino-based systems.

INSTALLATION
============

Library version:
Copy the PowerMonitor files and Examples folders to a new folder in your Arduino Libraries path
(e.g. /usr/share/arduino/libraries/PowerMonitor)
Restart the IDE, the examples and libraries should then become available.

there is one example sketch that queries basic AC parameters for th eline being monitored
